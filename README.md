## 这是什么？

这是[Dplayer播放器](https://github.com/MoePlayer/DPlayer)的弹幕服务器

## 为什么要做这个东西？

因为作者免费提供那个弹幕服务器貌似是挂掉了，看了一下开源的[源码](https://github.com/MoePlayer/DPlayer-node)，还需要`docker`，我看到这东西就烦，然后转手就自己撸了一个出来。
